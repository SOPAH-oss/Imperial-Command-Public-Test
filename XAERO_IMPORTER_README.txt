Xaero Waypoint Importer GUI

Run xaero_waypoint_importer_gui.bat from the same folder as the bot package.

What it imports:
- Walk waypoint -> waypoints.json
- Stasis pearl -> pearls.json
- Book writer chest -> ledger_chests.json
- Bank intake chest -> ledger_chests.json
- Butler source chest -> butler_chests.json
- Butler destination chest -> butler_waypoints.json

Use "Load Xaero File" and select Xaero's waypoints.txt. The GUI reviews each waypoint one by one so you can choose the destination type before it writes anything.
